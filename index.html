Progress Report For Insight Software

Phase 3: Advanced Front-End or Advanced Back-end

After an introduction to both front- and back-end developement in the first two phases, learners choose to focus on either front-end or back-end development in the third phase.

Our beginner back-end developers take on building APIs in Python using Django REST Framework. Front-end developers learn how to build single-page applications using React.

For the end-of-phase project, developers work on teams to create a single application, with a React front-end and a Django API back-end.
